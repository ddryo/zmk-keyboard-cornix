# Cornix LED インジケーター解説

`cornix_indicator` シールドで有効化される RGB LED（左右各2個）の表示内容。
ZMK モジュール [`zmk-rgbled-widget`](https://github.com/hitsmaxft/zmk-rgbled-widget) による実装。

## LED 配置（2-LED 共有モード）

```
┌───────┬───────┐
│ LED 0 │ LED 1 │
│ 🔋+📋 │ 📶+📋 │
│バッテリ│ 接続  │
└───────┴───────┘
```

レイヤー表示は両方の LED を一時的に借りて表示し、500ms 後に元に戻る。

## 起動時の点滅シーケンス

1. **バッテリー残量**
   - 🟢 緑: 80% 以上
   - 🟡 黄: 20〜80%
   - 🔴 赤: 20% 未満

2. **BT 接続状態**（左側のみ）
   - 🔵 青: 接続済み
   - 🟡 黄（パルス）: アドバタイジング中（ペアリング待ち）
   - 🔴 赤: 未接続

## 常時表示

| 状態 | 色 | パターン |
|------|-----|---------|
| BLE 接続済み | 🔵 青 | 点灯 |
| BLE アドバタイジング中 | 🟡 黄 | パルス（呼吸エフェクト） |
| BLE 未接続 | 🔴 赤 | 点滅（1000ms 間隔） |
| USB 優先中 | 🔵 シアン | 点灯 |
| バッテリー低下 | 🔴 赤 | 点滅（残量変化時） |
| バッテリー危険（5% 未満） | 🔴 赤 | 高速点滅（250ms 間隔、常時） |

## 左右の表示内容の違い

| 情報 | 左（central） | 右（peripheral） |
|------|:---:|:---:|
| バッテリー残量 | 自分の | 自分の |
| ホストとの BT 接続 | ✅ | ❌ |
| 左右間の split 接続 | ❌ | ✅ |
| BT プロファイル切替 | ✅ | ❌ |
| レイヤー状態 | ✅ | ❌ |
| USB 優先表示 | ✅ | ❌ |

右側（peripheral）は左側との split 接続状態と自身のバッテリー残量のみ表示。

## BT プロファイル切替時

func_layer で `BT_SEL 0/1/2` を押すと、左側で接続状態を再表示する。

## 関連設定

- `build.yaml`: `shield: cornix_indicator` で有効化
- `cornix_indicator.conf`: 明るさは `CONFIG_RGBLED_WIDGET_BRIGHTNESS=64`（0〜255）
